Petri net model 
=================

_This is lab for RSREU_

Used:
* gulp-babel-browserify
* es6-bootstrap-jQuery
* MVC :unamused:  

### Installation

```bash
git clone https://github.com/rodinwow/petri-net-model.git
cd petri-net-model
npm install
gulp
npm start
```
